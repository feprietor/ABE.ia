# ABE — Agricultural Business Expert · Support Website

Official support and landing page for **ABE**, an AI-powered agricultural advisory system built for beginning Iowa farmers. Developed by **SAU Hive Mind** for the Pi515 AI Challenge 2026.

---

## About

ABE helps beginning Iowa farmers navigate financial decisions, program eligibility, and production economics through a Telegram bot interface backed by verified Iowa-specific data sources (ISU Extension, USDA NASS/AMS, Iowa Finance Authority).

This repository contains the **static support website** — a single-file HTML page that introduces ABE, explains its capabilities, and connects visitors to the bot.

## Features

- Intro / Hero section with live conversation example
- About, Skills, How It Works, Ethics, Team, FAQ, and Contact sections
- Fully responsive — optimized for desktop, tablet, and mobile
- Contact form powered by [EmailJS](https://www.emailjs.com) — no backend required
- Direct links to the ABE Telegram bot (`t.me/the_ABEbot`)
- Hamburger navigation for small screens

---

## Setup

No build tools or dependencies. Just open `abe_support.html` in a browser.
